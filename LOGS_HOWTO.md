# 🔍 Как посмотреть логи биометрии на телефоне

## Просто и быстро (без компьютера):

### 1. Откройте приложение в Telegram
   - Запустите ATS WALLET

### 2. Откройте консоль разработчика
   - Профиль → Настройки
   - Прокрутите вниз до раздела "О приложении"
   - Нажмите **"Консоль разработчика (логи)"**

### 3. Посмотрите логи
   - Откроется мобильная консоль (Eruda)
   - Перейдите на вкладку **Console**
   - Все логи с `[Biometry]` будут здесь

### 4. Проверьте биометрию
   - Закройте консоль (крестик)
   - Профиль → Безопасность → Включить защиту → Биометрия
   - Снова откройте консоль через Настройки
   - Посмотрите логи

## Что искать:

### ✅ Если работает:
```
[Biometry] ✅ BiometryManager available
[Biometry] Final info: { available: true, type: "finger" }
[Biometry] ✅ Biometry AVAILABLE
```

### ❌ Если не работает:
```
[Biometry] ❌ isBiometricAvailable = FALSE
[Biometry] Possible reasons:
  - Biometry not set up in device settings
```

## Возможные проблемы:

### 1. `available: false` (самая частая)
**Решение:**
- Настройки телефона → Безопасность → Отпечаток пальца
- Добавьте отпечаток если не настроено
- Перезагрузите телефон

### 2. `BiometryManager not present`
**Решение:**
- Обновите Telegram до последней версии
- Минимум 7.2, у вас 9.1 ✅

### 3. `Version too old`
**Решение:**
- Обновите Telegram
